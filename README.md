# PureNetPI
We plan to create a Pi-hole using a Raspberry Pi to implement a network-wide ad-blocking solution. The process involves setting up a Raspberry Pi with the Raspberry Pi OS, followed by installing Pi-hole, a DNS sinkhole and firewall that filters out advertisements and malware domains at the network level. By configuring the home router to use the Raspberry Pi as its primary DNS server, all DNS requests from devices on the network will pass through the Pi-hole. This will effectively block unwanted ads and enhance privacy and security across the network without requiring any modifications on individual devices.
